LAYER 0 - METADATA

Layer 0 defines transparency and sprite configuration. Fill ONLY column 0.

Metadata cells (column 0 only):
  • [0,0] = angle count (digit 1-8). If missing = 1 angle.
  • [height×1, 0] = animation 1 frame count
  • [height×2, 0] = animation 2 frame count
  • All OTHER cells = your color key (e.g., magenta)

Transparency

  • Layer 0 bg color = transparency key
  • In Layer 2: if fg or bg matches Layer 0's key = transparent
  • Magenta (255,0,255) = fully transparent

Height Encoding (Layer 1)

  • '0'-'9' = heights 0-9
  • 'A'-'Z' or 'a'-'z' = heights 10-35
